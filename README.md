## AMAZON SALES ANALYSIS DASHBOARDS 
 The Project present an End-To-End Amazon sales dashboards using Power BI, SQL objective is to Analysed Amazon sales data, identify Business trends and build an interactive dashboard that helps understand revenue, orders and productions sales 

## Project Overview 

The Amazon sales dashboards project focuses on transforming row data into meaningful insights using data cleaning, SQL analyst, and visualization techniques.
 •	The dashboards highlights:
 •	Sales performance across different subcategory
 •	Revenue tends over time 
 •	Customer order patterns
 •	Best and worst performing categories and subcategories 
 This project demonstrates real-world data analyst workflow:
## Raw Data → Data Cleaning → SQL Analysis → Visualization

# Data cleaning process
•	Data was cleaned using Power Query in Power BI:
•	Removed null values
•	Corrected data types
•	Standardized column names
•	Removed duplicates
•	Created calculated columns

## SQL Analysis 
SQL Queries were used to calculate KPIs and trends such as:
•	Total revenue 
•	Average Order value
•	Sales by category 
•	Sales by years, months 
•	Sales by subcategory
